# Tic-Tac-Toe CLI

A simple command-line Tic-Tac-Toe game for two players.

## Features

- Play Tic-Tac-Toe in the terminal
- Two-player mode
- Displays board state after each move
- Simple input prompts

## Requirements

- A C++ compiler such as g++.

## Installation

1. Clone or download the repository
2. Navigate to the project folder

## Usage

- Compile and run the program from the terminal
- Follow on-screen prompts to enter moves
- Enter row and column numbers to place X or O
